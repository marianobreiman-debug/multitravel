V1 · Capa condición

Estructura base + capa estándar reusable para restricciones

Cada beneficio mantiene su lectura principal. Si aplica, aparece debajo una línea de condición con patrón visual siempre igual: ícono + texto corto. El mismo patrón sirve para cualquier futura restricción del sistema.

  • Comafi 6 CSI → condición verde: "Cartera habilitada"
  • Comafi 9 CSI → condición ámbar: "Solo BINES habilitados"
  • Misma UI, semántica diferente

V2 · Tres capas

Arquitectura interna fija: beneficio → alcance → condición

Cada ítem tiene siempre la misma estructura. La condición es la tercera capa, visible solo cuando aplica. Patrón consistente y semánticamente claro para escalar a cualquier caso.

  • Capa 1: cuotas + tipo (dominante)
  • Capa 2: banco(s) / alcance
  • Capa 3: condición especial (solo si restringe)

V3 · Bloques modulares

Cada promo es un módulo con campos formales definidos

La condición deja de ser una notita al pie y pasa a ser una sección propia del bloque. Escala a cualquier acuerdo complejo: la excepción tiene el mismo peso estructural que el beneficio.

  • Campo "condición" visible en el módulo, no como subnota
  • Bloques especiales con tratamiento visual diferenciado
  • Apto para casuística variable sin lista plana

V4 · Aplicabilidad

Separación visual por nivel de universalidad

El usuario entiende de un vistazo qué aplica a todos, qué depende del banco, y qué tiene restricción especial. Comafi 9 CSI solo puede estar en "Condiciones especiales" — estructuralmente imposible confundirlo con los 6 CSI.

  • Beneficios generales → para todos
  • Beneficios por banco → cartera específica
  • Condiciones especiales → BIN / segmento / vigencia

V5 · Etiquetas

Taxonomía de estados — capa transversal aplicable a cualquier variante

No reemplaza la estructura, la complementa. Cada promo tiene una etiqueta que comunica su nivel de universalidad de forma inmediata y estandarizada.

  • Verde: todos los bancos
  • Azul: banco adherido / cartera
  • Oscuro: solo segmento Black
  • Ámbar: BINES habilitados
  • Naranja: vigencia especial
